# Mides Global Realtors

A modern, lightweight real estate website for Mides Global Realtors in Lagos, Nigeria.

## Features

- 🏠 Property listings (lands & buildings)
- 🤖 Smart chatbot with local knowledge base
- 📱 Mobile-first responsive design
- 🎨 Smooth animations with Framer Motion
- 💬 WhatsApp integration for lead capture
- 🎯 Lead generation popups
- ⚡ Fast performance with Next.js 15

## Tech Stack

- **Frontend:** Next.js 15, Tailwind CSS, Framer Motion
- **CMS:** Sanity.io (coming soon)
- **Deployment:** Vercel
- **Chatbot:** Custom with Fuse.js fuzzy search

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/mides-global-realtors.git

# Install dependencies
npm install

# Run development server
npm run dev
